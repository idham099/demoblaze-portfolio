# demoblaze-portfolio

1. Nama Proyek: DemoBlaze Full-Stack Automation Portfolio
   "Ini adalah proyek Full-Stack (UI, API, Performance) "
   
2. Tujuan: Membangun Automation Framework E2E yang stabil
   Menyoroti penggunaan CI/CD dan pengujian Non-Fungsional (K6).
   
3. Framework: Cypress (JavaScript/TypeScript), Page Object Model (POM). API: Cypress cy.request() & cy.intercept() / JSONPlaceholder. Performance: K6. CI/CD: GitHub Actions
  
4. Strategi Pengujian: Menggunakan API Mocking/Stubbing untuk menstabilkan test case UI, untuk menunjukkan kemampuan dalam menangani Dependensi dan Arsitektur Testing.
5. Instruksi:
   npm install
   npx cypress open
   perintah k6 run
  
6. Link ke hasil CI/CD GitHub Actions (Status Badge) dan Test Summary Report (Allure/HTML Report)
